# G-watch

ガンダムシリーズの作品を記録・レビューできる Web アプリです。

---

## 概要
- ガンダム作品の一覧・詳細表示
- 作品ごとのレビュー投稿
- ログインユーザーのみレビュー投稿可能
- 平均評価・初心者向けおすすめ度の表示

ポートフォリオ用の個人開発アプリです。

---

## 主な機能
- ユーザー登録 / ログイン / ログアウト
- 作品（Work）のCRUD
- レビュー投稿（視聴予定 / 視聴済み、評価、コメント）
- ログイン制御による投稿制限

---

## 使用技術
- Ruby 3.x
- Ruby on Rails 6.1
- PostgreSQL
- HTML / ERB / SCSS
- セッション認証（has_secure_password）

---

## 工夫した点
- レビュー投稿時に `current_user` を使用し、user_id をフォームから渡さない設計
- 未ログイン時は投稿フォームを非表示にし、ログイン導線を表示
- 作品詳細ページでレビュー平均を自動計算
